Design Choices:

1. Abstract Factory: Utilizado para criar diferentes tipos de produtos baseados em categorias específicas (eletrônicos e roupas).
2. Injeção de Dependência: Implementado um contêiner simples para injeção de dependência para manter o código desacoplado e flexível.
3. Gerenciamento de Inventário: Criado um gerenciador de inventário que utiliza a fábrica abstrata e a injeção de dependência para gerenciar produtos.

Como executar:

1. Compile todas as classes.
2. Execute a classe Main.
